General Info

A while ago I decided to learn all about API's and how to use them. So as my practice with them I decided to use the Spotify API which has some nice documentation to learn what it can do and I thought it may be interesting for users to see information related to their Spotify.

I encountered quite a few difficulties through this project such as using tokens and credentials to be able to access the info within the API, though now after having built it I feel like I have learnt a lot and an important programming skill.
